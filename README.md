> **1.1 million Americans ration insulin.** 600+ manufacturer patient assistance programs, state programs, 340B pharmacies, and copay cards exist — but the application process is Byzantine. Different forms, eligibility criteria, renewal schedules. The complexity is the barrier, not the availability.

## The Solution

Intake agent collects insurance, income, medications, diagnoses. Matches against ALL programs. Auto-fills applications. Tracks renewal deadlines. Alerts when generics become available. Cost comparison across pharmacies.

## Agents

**IntakeAgent** — Collect patient details and current medications
**ProgramMatcher** — Match against 600+ PAPs, state programs, 340B pharmacies, manufacturer copay cards
**ApplicationFiller** — Auto-generate and pre-fill program applications
**RenewalTracker** — Track deadlines, alert before expiration
**CostComparer** — Compare prices across pharmacies (GoodRx-style)
**GenericFinder** — Identify cheaper equivalent medications via FDA Orange Book

## Safety

MANDATORY: This is not medical advice. Always consult your pharmacist or physician before changing medications.

## Data Sources

RxAssist database, NeedyMeds directory, FDA Orange Book, GoodRx pricing, state Medicaid formularies, openFDA drug data
